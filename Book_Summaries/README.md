## 📝 How to Contribute

1. **Read a book** on ethical hacking, security, or related topics.
2. **Create a summary** in Markdown format (or PDF if preferred).
3. **Include:**
   - Key concepts and takeaways
   - Practical lab exercises or commands
   - Real-world applications
   - Your personal notes or critiques
4. **Submit a PR** or add the file directly if you have write access.
